# Desktop dotfiles

credit to torrinfail and luke smith for some of the configurations

- dwm (window manager)
- st (terminal)
- dwmblocks (status bar)
- mpc/ncmpcpp (music)
- mpv (video)
- pcmanfm, ranger (file manager)
- vim (text editor)
- firefox (browser)
- zathura (document reader)
- qbittorrent (torrent client)
- paru (aur helper)
